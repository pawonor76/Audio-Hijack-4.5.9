# What Is Audio Hijack?

Audio Hijack is a professional-grade audio recording and processing application designed exclusively for macOS.

Unlike traditional recording software that focuses only on microphones or audio interfaces, Audio Hijack can capture sound from:

* Individual applications
* Entire system audio
* Web browsers
* Streaming services
* VoIP applications
* External microphones
* USB mixers
* Audio interfaces
* Virtual audio devices

The software uses a visual block-based interface that allows users to build custom audio workflows. Sources, effects, processors, recorders, and outputs are connected together in a simple drag-and-drop environment.

This flexibility has made Audio Hijack popular among podcasters, broadcasters, musicians, journalists, educators, and content creators.

# What's New in Audio Hijack 4.5.9?

Audio Hijack 4.5.9 introduces several notable improvements.

### Quo VoIP Capture Support

The biggest addition is support for Quo, formerly known as OpenPhone. Users can now capture both sides of conversations directly from the application. This is especially useful for business users, podcasters, and journalists who conduct interviews remotely.

### Broadcast and Live Stream Fixes

A regression introduced in version 4.5.8 prevented certain Broadcast and Live Stream settings from being saved properly. Version 4.5.9 resolves this issue and restores expected behavior.

### Interface Improvements

Minor graphical issues affecting streaming indicators have been corrected, improving the overall user experience.

### Updated macOS Requirements

Audio Hijack 4.5.9 now requires macOS 14.4 or later, ensuring compatibility with Apple's latest audio technologies and frameworks.
